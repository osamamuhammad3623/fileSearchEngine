# File Search Engine

## Features
Search for a keyword in:
- Directory names
- File names
- In-file text

Supported file extensions:
- headerfiles (.h)
- C/C++ files (.c/cpp)
- Python files (.py)
- Text files (.txt)

## Framework
- Qt 5 with C++17
